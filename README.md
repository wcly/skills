# Skills 技能集合

本目录包含一系列自定义技能，用于增强你的 AI 助手能力。

## 技能列表

### 1. article-checker
**文章智能校验工具**

基于 LLM 的文章校对助手，可以检测并修复：
- 错别字
- 病句
- 拼写错误
- 逻辑错误
- 矛盾冲突
- 语句不通顺

**触发词**：文章校验、检查文章、校对文章、文章查错

---

### 2. juejin-auto-checkin
**稀土掘金自动签到**

自动化完成稀土掘金网站的以下任务：
- 每日自动签到（获得矿石奖励）
- 免费单抽（每天一次抽奖机会）
- 定时任务支持

**触发词**：稀土掘金签到、掘金抽奖、自动签到、免费单抽

---

### 3. mermaid-diagram-generator
**Mermaid 图表生成器**

将自然语言描述转换为 Mermaid 图表代码，支持：
- 流程图 (Flowchart)
- 时序图 (Sequence Diagram)
- 类图 (Class Diagram)
- 状态图 (State Diagram)
- ER 图 (ER Diagram)
- 甘特图 (Gantt Chart)
- 饼图 (Pie Chart)
- 思维导图 (Mind Map)

**触发词**：创建流程图、画图、时序图、Mermaid

---

### 4. skill-security-checker
**技能安全检查器**

在下载或使用 Skill/代码仓库前进行安全风险评估：
- 恶意代码检测
- 敏感信息窃取检测
- 仓库信誉度评估
- 可选：运行时行为监控

**触发词**：检查安全性、扫描恶意代码、验证代码安全

---

## 使用方法

每个技能都有独立的触发词，直接告诉 AI 你想要做什么即可激活对应的技能。

例如：
- "帮我校对这篇文章：..."
- "执行稀土掘金签到"
- "画一个用户登录的流程图"
- "检查这个仓库的安全性"
